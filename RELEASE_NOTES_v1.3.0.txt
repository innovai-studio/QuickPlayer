<en-US>
QuickPlayer v1.3.0 Major Update!

New:
• Stable background playback — music keeps going when the screen locks, with lockscreen / notification / Bluetooth headset controls
• Collapsible panels — Speed / Pitch / Focus EQ / Metronome / A-B Loop / Markers can each be hidden to avoid mis-taps; the expand/collapse state is remembered per panel
• Rewritten metronome audio path — native Android SoundPool, mixes alongside the song instead of fighting for audio focus
• Steadier A-B Loop — no longer routed through ClippingAudioSource, which conflicted with the MediaSession

Improvements:
• Notification / lockscreen show the song name instead of the file path
• Pressing play after a track ends restarts cleanly with one tap
• Metronome no longer gets stuck when the loop wraps back to A

Continuing:
• Focus EQ with 5 presets + custom EQ + live spectrum analyser
• Tap-to-sync metronome with multi time signatures (2/4 / 3/4 / 4/4 / 6/8 / 12/8)
• Speed 0.25x–2.0x, pitch ±12 semitones
• A-B loop and practice markers
</en-US>
